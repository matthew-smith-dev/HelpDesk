# Help Desk Interview Notes

## How to explain this project

Help Desk is a simple help desk ticketing system. It allows a support team to record user issues, prioritise them, track progress, and document work notes.

## Why this is useful in IT support

Help desk teams need a system to avoid losing user requests. Tickets help track who reported the issue, what the issue is, how urgent it is, and what troubleshooting steps were taken.

## What I learned

- How ticket workflows operate
- How to model tickets, comments, status, and priority in C#
- How to build a simple ASP.NET Core web app
- How to store and load data locally
- How to document a technical project clearly
